# Project Overview

This project analyzes pharmacy sales data using **MySQL** and **Power BI**.  
The dataset contains 1500+ records across 5 relational tables:

- Customers
- Products
- Stores
- Employees
- Sales

The goal of this project is to:

- Clean and structure the data in MySQL  
- Create SQL analysis queries  
- Model the data using relationships  
- Build a multi-page Power BI dashboard  
- Generate insights to support business decisions
